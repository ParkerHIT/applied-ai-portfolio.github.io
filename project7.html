Cross-diagnostic screening across autism, neurodevelopmental, and psychiatric disorder populations through statistically validated multimodal ML pipeline

Overview

Female autism is significantly underdiagnosed and misdiagnosed in clinical practice, with females often presenting with different behavioral phenotypes than males. Traditional diagnostic approaches fail to capture female-specific patterns, leading to missed diagnoses or misclassification as ADHD, anxiety, mood disorders, or other psychiatric conditions.

Designed and built statistically validated multimodal ML pipeline integrating behavioral demographics and resting-state fMRI connectivity features to identify autism biomarkers in females through A/B testing of feature modalities and connectivity networks. Applied trained model across multiple independent datasets for cross-diagnostic screening: ADHD-200 (neurodevelopmental disorders) and UCLA Consortium for Neuropsychiatric Phenomics (psychiatric disorders including Bipolar, Schizophrenia, and ADHD), identifying autism-signature patterns across diverse diagnostic populations.

Implementation

Data Preprocessing

Implemented robust data cleaning pipelines for ABIDE (Autism Brain Imaging Data Exchange), ADHD-200, and UCLA Consortium for Neuropsychiatric Phenomics (CNP) datasets. Handled missing values through median imputation for demographics and careful NaN cleaning for neuroimaging data. Extracted ROI timeseries from resting-state fMRI across multiple sites, harmonizing data from different scanner protocols and acquisition parameters.

Feature Engineering

Designed atlas-agnostic network-level connectivity features enabling seamless application across different neuroimaging parcellations. Extracted within-network connectivity metrics (mean, std, max) for Default Mode Network (DMN), Salience Network (SN), and Frontoparietal Network (FPN). Computed inter-network connectivity patterns and graph theory metrics including edge density and node degree distributions.

Statistical Validation

Implemented comprehensive A/B testing framework with paired statistical tests (t-test + Wilcoxon signed-rank) to systematically compare feature modalities and network configurations. Validated biomarker candidates using Mann-Whitney U tests (p < 0.05) with effect size calculations (Cohen's d). Prioritized scientific rigor over performance optimization, ensuring reproducible and clinically meaningful results.

Model Training & Validation

Trained XGBoost classifier selected for interpretable feature importance metrics and built-in regularization to prevent overfitting on small sample (N=157). Employed stratified 5-fold cross-validation to ensure robust performance estimation across different data partitions. Extracted feature importance rankings and validated top biomarkers through independent statistical tests.

Cross-Diagnostic Application

Applied trained model to multiple independent datasets: ADHD-200 (neurodevelopmental disorders, 59 subjects) extracting 23-feature set from ROI 1000 atlas, and UCLA CNP (psychiatric disorders including Bipolar, Schizophrenia, and ADHD, 54 subjects). Adapted pipeline to handle varying feature availability through partial feature retraining using age and connectivity features only (20 features), enabling application across datasets with different phenotypic assessments. Generated autism probability scores for each subject, flagging top 20% as high-risk cases exhibiting autism-signature patterns warranting clinical review.

Key Technical Decisions

01 — Atlas-Agnostic Network Features

Designed network-level connectivity features (DMN, SN, FPN) that generalize across different brain parcellations, enabling successful transfer from ABIDE's CC200 atlas (200 ROIs) to ADHD-200's ROI 1000 atlas (1000 ROIs) without model retraining. This architectural choice prioritized cross-dataset applicability over atlas-specific optimization.

02 — Biomarker-Based Approach

Deliberately excluded diagnostic instruments (ADOS, ADI-R) from feature set despite their high predictive power to avoid data leakage and enable true biomarker discovery. This decision prioritized scientific validity and cross-diagnostic applicability over performance metrics, as diagnostic instruments are unavailable in non-autism populations.

03 — Statistical Rigor Over Performance

Implemented comprehensive statistical validation (A/B testing, cross-validation, biomarker validation) rather than aggressive hyperparameter tuning. Accepted realistic performance metrics while ensuring reproducible methodology and honest reporting of results, contributing to more reliable medical machine learning practices.

04 — Adaptive Feature Selection

Designed pipeline to accommodate varying feature availability across datasets through partial feature retraining. When applying to UCLA CNP psychiatric disorder population lacking IQ assessments, retrained model using only age and connectivity features (20 features), demonstrating methodological flexibility while maintaining scientific rigor. This approach enables cross-diagnostic screening across datasets with different phenotypic assessment protocols.

Successfully developed statistically validated multimodal ML pipeline through rigorous A/B testing framework. Validated pipeline through 5-fold cross-validation, demonstrating reproducible results across training folds while maintaining scientific rigor and avoiding overfitting.

Successfully applied pipeline to cross-diagnostic screening across multiple independent datasets. ADHD-200 screening (59 subjects) identified 20% with autism-signature patterns. UCLA CNP psychiatric disorder screening (54 subjects: Bipolar, Schizophrenia, ADHD) flagged 20% for clinical review, with notable clinical finding that Bipolar disorder subjects showed highest autism-signature prevalence (33.3% flagged) compared to ADHD (14.3%) and Schizophrenia (8.3%), suggesting significant phenotypic overlap between autism and bipolar disorder. Top-scoring cases exhibited autism probabilities up to 98%.

Biomarker validation identified verbal IQ as statistically significant discriminator (p=0.03, Cohen's d=-0.37). A/B testing revealed no significant differences between modalities (p=0.60) or network configurations (p=0.63), validating selection of multimodal three-network approach based on neuroscientific rationale rather than performance optimization alone.

Clinical Utility

Pipeline generates interpretable autism probability scores suitable for clinical review, providing objective data to supplement diagnostic assessment. Identified high-risk cases across neurodevelopmental (12 ADHD-200 cases) and psychiatric populations (11 UCLA CNP cases including Bipolar, Schizophrenia, and ADHD), enabling clinicians to prioritize subjects for comprehensive autism evaluation across multiple diagnostic categories and potentially identify misdiagnosed individuals.

Methodological Contributions

Demonstrated feasibility of cross-diagnostic screening using autism-trained models on independent neurodevelopmental populations. Established reproducible A/B testing framework for feature selection validation. Contributed to honest reporting practices in medical ML by prioritizing scientific validity over inflated performance metrics.

Psychiatric Disorder Screening

Extended screening to UCLA Consortium for Neuropsychiatric Phenomics (CNP), applying pipeline to 54 females with psychiatric disorders (21 Bipolar, 12 Schizophrenia, 21 ADHD). Identified 20% exhibiting strong autism-signature patterns (mean probability: 0.77), with clinical finding that Bipolar disorder subjects showed highest autism-signature prevalence (33.3% flagged) compared to ADHD (14.3%) and Schizophrenia (8.3%). This pattern suggests significant phenotypic overlap between autism and bipolar disorder, indicating potential value of autism assessment in bipolar populations.

Impact

Developed data-driven approach to identify potential misdiagnosed cases in underrepresented female populations where traditional diagnostic pathways often fail. Pipeline provides objective, reproducible screening tool to supplement clinical assessment and address diagnostic disparities.

Demonstrated cross-diagnostic screening framework across autism, neurodevelopmental, and psychiatric disorder populations (3 independent datasets, 270 total subjects), establishing methodology for identifying shared phenotypic patterns across diverse diagnostic categories including Bipolar disorder, ADHD, and Schizophrenia. This approach enables systematic flagging of cases for clinical re-evaluation and potential diagnostic revision across multiple psychiatric and neurodevelopmental conditions.

Implemented comprehensive statistical validation framework establishing best practices for medical ML research. Prioritized reproducibility and scientific rigor over performance metrics, contributing to more reliable and clinically meaningful machine learning applications in healthcare.
